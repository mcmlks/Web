マイクラ PlayStation 参加方法

PS4 / PS5 の Minecraft からサーバーに参加するための設定手順です。

目次
  1. PlayStation のネットワーク設定
  2. DNS 設定の変更
  3. マインクラフトから接続

① PlayStation のネットワーク設定

まずは PS 本体のネットワーク設定を開き、インターネット接続の設定を「カスタム」で進めます。

  1. ホーム画面から 「設定」 を開きます。
  2. 設定メニューから 「ネットワーク」 を選択します。
  3. 「インターネット接続を設定する」 を開きます。
  4. 接続方法の選択画面で 「カスタム」 を選びます。
  5. それ以外の項目（IPアドレス設定など）は、基本的に 「自動」や「推奨」など上の選択肢 を選んで問題ありません。
お使いの PS4 / PS5 のバージョンによって、画面の表示やボタン名が少し異なる場合がありますが、
「設定」→「ネットワーク」→「インターネット接続を設定する」→「カスタム」という流れは共通です。

② DNS 設定を変更する

DNS 設定だけ「手動」に切り替え、指定された数値を入力します。

  1. 設定項目の中の 「DNS 設定」 を 「手動」 にします。
  2. 表示された DNS の値を、次のように入力します。
プライマリー DNS
45.55.68.52

プライマリー DNS は必ずこの値を入力してください。

セカンダリー DNS
8.8.8.8

以下の値でも代用できます：

1.1.1.1 9.9.9.9
設定画面のレイアウトが違う場合は、PS 本体のヘルプや公式ガイドもあわせて確認してください。
うまくいかないときは、いったん DNS をもとの設定（自動など）に戻してやり直すと解決することがあります。

③ マインクラフトからサーバーに接続

DNS 設定が完了したら、Minecraft を起動してサーバーに参加します。

  1. PlayStation で マインクラフト を起動します。
  2. タイトル画面から いずれかの特集サーバー（Featured Server） を選んで開きます。
    （どのサーバーを選んでも構いません）
  3. サーバー選択画面で 「connect server」 を選択します。
  4. 表示された入力欄に、次の情報を入力します。
    • 上段：サーバーアドレス（IPアドレス / ドメイン）
    • 中段：ポート番号
    • 下段：自分のデバイスに表示される名前
  5. 「Add to server list」 にチェックを入れます。
  6. 一番下の 「送信」ボタン を押すと、サーバーがリストに追加されます。
  7. サーバー一覧から追加したサーバーを選ぶと、参加できます。
  • 最初から入力されている数字は、そのまま上書きして構いません。
  • 名前欄では 漢字・ひらがな・カタカナは使用できません。英数字のみ使用できます。
接続できない場合は、DNS 設定が正しいか・ネットワークが安定しているか確認し、
本体やゲームを再起動してから再度お試しください。
